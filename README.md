# System Szkoleń - Projekt Django

Projekt został utworzony na potrzeby realizacji i zaliczenia laboratorium z przedmiotu AJP. 

## Opis projektu
Aplikacja internetowa oparta na frameworku Django, służąca do przeglądania i zarządzania ofertą szkoleniową. System składa się z następujących modułów:
* **Oferta publiczna** – przeglądanie dostępnych kategorii i przypisanych do nich szkoleń.
* **Rejestracja** – formularz zgłoszeniowy pozwalający zapisać się na wybrane szkolenie.
* **Panel Administratora** – zarządzanie bazą danych (dodawanie/edycja kategorii, szkoleń oraz przeglądanie rejestracji).
* **API** – endpointy zwracające dane z bazy w formacie JSON.
